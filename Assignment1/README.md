# CE-ESY-assignments
قمنا بتعريف CircularBuffer ليحتوي على مصفوفة لتخزين البيانات،
ومؤشرين: head للقراءة .. و tail للكتابة .. ومتغير count لمتابعة عدد العناصر.
دالة init : تقوم بضبط جميع المؤشرات والعدادات إلى الصفر عند بدء البرنامج.
يُعتبر المخزن ممتلئاً إذا تساوى count مع الحجم الكلي size.
...................
الكتابة: عند إضافة حرف، يتم وضعه في مكان tail ثم يتحرك المؤشر خطوة للأمام >> واستخدمنا (tail + 1) % SIZE لضمان عودة المؤشر للبداية (دائرياً) عند الوصول لنهاية المصفوفة.
القراءة: عند قراءة حرف، نأخذه من مكان head ثم يتحرك المؤشر باستخدام نفس المنطق الدائري (head + 1) % SIZE.
البرنامج يقوم بدمج السلسلة النصية " CE-ESY" مع الاسم المدخل قبل تخزينه للتأكد من معالجة السلسلة كاملة داخل المخزن.
